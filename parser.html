Info

Write your instructions without ';' at the end and ',' between values, and use the value of the register (0 -> 31) to assign the register you want.
Example of instruction:
=> addi 20 22 -152
instead of:
=> addi $s4, $s6, -152;


Comments ARE allowed!

'&&' for comment line.
Example:
=> addi 20 22 -152 && Should have 32 in register 20
The part "&& Should have 32 in register 20" will be ignored.

'*&' for beginning of comment block and '&*' for the end of it.
Example:
=> addi 20 22 -128 *& Should have 32 in register 20
